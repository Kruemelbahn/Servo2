# Servo2

Servo2 is the possibility, to control two Servos without digital knick-knacks.

For more details please refer to the documentation.<br>

The software [ServoAnsteuerung](https://www.github.com/Kruemelbahn/ServoAnsteuerung) can be used to change parameters in hex-file.<br>

The PCB fits into 'Merscheider Schacht', corresponding informations can be found on my [Website](https://kruemelsoft.hier-im-netz.de/schacht.htm)
or as an [Infoletter](https://github.com/Kruemelbahn/Infoletter/blob/main/Kr%C3%BCmelbahn%20Info%2012%20-%20Merscheider%20Schacht.pdf)
